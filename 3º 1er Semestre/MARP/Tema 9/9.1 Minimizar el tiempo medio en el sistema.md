- Objetivo: Minimizar el tiempo que se tarda en hacer todas las tareas teniendo en cuneta el tiempo de espera de cada una así como lo que tarda en realizar la tarea
- Características:
	- Hay N tareas
	- Cada tarea tiene un tiempo de uso que es ti
	- Ejemplo n = 3 --> t1 = 5, t2 = 10, t3 = 3
		- Si se ejecutan en orden (t1,t2,t3) el tiempo es 5 + (5+10) + (5+10+3) = 38
		- Pero si el orden es 3,1,2 sale 29
- Planificación óptima:
	- Aquí no hay conjuntos, ya que se han de coger todas y la respuesta consiste en encontrar una ordenación de las tareas hasta encontrar el menor tiempo o solución óptima
	- La forma de resolver este problema es ordenar las tareas de menor a mayor tiempo, de esta forma se garantiza el mínimo posible
