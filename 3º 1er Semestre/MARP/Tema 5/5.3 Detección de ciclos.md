- Un ciclo dirigido es un camino dirigido de tamaño a que cumple que v0 = va
- Para detectar estos ciclos se utiliza un recorrido en profundidad o dfs del grafo dirigido
- Aquí aparte de comprobar si el vector ya estaba visitado hay que comprobar si el vértice no está apilado, es decir, los vértices pendientes se almacenan en un vector de bool y si durante el recorrido de uno se llega a un vértice que siga pendiente apilado = true entonces tenemos un ciclo
- Implementación:
	- Se puede construir el ciclo partiendo del vértice v del dfs hasta llegar al vértice w que está apilado preguntando por el anterior y agregandolo a una pila llamada ciclo. Finalmente metemos w y v
	- Se añade un booleano hayCiclo para cortar bucle de dfs y el vector de bools de apilados para la detección de estos ciclos
	- Si queremos construir el ciclo es esencial
		- Visitados
		- Anteriores
		- Ciclo
		- hayCiclo
	 - En caso contrario
		 - Visitados
		 - hayCiclo
- Complejidad de O(V + A)
