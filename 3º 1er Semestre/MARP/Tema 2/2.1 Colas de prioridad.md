- Colas de prioridad:
	- Estas colas se en un principio se ordenan en orden de llegada (FIFO), 
	- Estas colas se pueden ordenar gracias a los operador <, en el que la prioridad siempre viene dada por a < b o b < a
	- 2 tipos de colas de prioridad:
		- Cola mínimos: 
			- El elemento más prioritario es el elemento menor
		- Cola de máximos:
			- El elemento más prioritario es el elemento mayor
	- TAD:
		- Crear a cola vacía O(1)
		- Push (para insertar el elemento) O(log N)
		- top() devuelve el más prioritario O(1)
		- pop() elimina el más prioritario O(log N)
		- empty si la cola está vacía O(1)
		- size la cantidad de elementos O(1)
		- En C++ este TAD implementa la cola de máximos en todos los casos, no obstante si se utiliza el operador > en lugar del < este se convierte en cola de mínimos ya que devolvería el último elemento (más prioritario) que sería el equivalente al menor
		- Se construye de la siguiente manera:
			- priority_queue<TIPO, vector TIPO>, operador TIPO> myPrioQueue
	- USOS:
		- Como se utilizan los operadores < y > se puede trabajar con estructuras propias siempre que se definan estos operadores con bool operator<(other){return a < other} por ejemplo
