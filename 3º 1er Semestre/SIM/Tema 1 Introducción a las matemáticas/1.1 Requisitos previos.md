
- 1.1.1 Magnitudes físicas
	- Una magnitud es una propiedad física que puede ser medida de algina manera, destacamos magnitud escalar y magnitud vectorial
	- Vector: Entidad matemática que representa magnitudes vectoriales, es un array
	- Sistemas de coordenadas: Punto de origen y vectores unitarios

- 1.1.2 Vectores
	- Se interpreta con un módulo  y una dirección a través de la normalización de un vector unitario (n)
		- Módulo (a) = ||a|| = d = x^2 + y^2 + z^2 Todo en raíz
		- Dirección/módulo del vector
	
	 - 1.1.2.1 Operaciones
		 - Escala: Se cambia el módulo del vector sin cambiar la dirección (a = d x b) siendo a y b vectores 
		 - Normalización:   n = a/d
		 - Suma y Resta
		 - Producto Escalar: Multiplicación de 2 vectores cuyo resultado es un escalar:
			- Nos permite conocer el ángulo entre vectores
			- a * b = ||a|| * ||b|| * cos(θ) = ax * bx + ay * by + az * bz
			- Si a y b están normalizados el producto escalar es el coseno del ángulo que forman
			- Si se trata de posiciones podemos saber si una posición está detrás de otra
		- Producto Vectorial: Multiplicación de 2 vectores cuyo resultado es otro vector:
			- El vector resultante es perpendicular a los otros 2
			- No se hace en R2
			- No se usa para calcular ángulos 
			- No es conmutativo
			- Si a y b están normalizados el módulo del producto vectorial es el seno del ángulo que forman ambos vectores
			- ||a × b|| = ||a|| * ||b|| * sin(θ)
			- USOS:
				- Construir sistemas de coordenadas
			
-  1.1.3 Base Ortonormal:
	- Son tres vectores perpendiculares entre sí de módulo unidad
	- Se crea a través del producto vectorial

- 1.1.4 Matrices:
	- Se usan para almacenar la información espacial
	- Matriz traspuesta (se cambian filas por columnas)
	- Sumar y restar matrices (mismas dimensiones)
	- Multiplicar matriz * número (Multiplicar cada elemento de la matriz por ese número)
	- Multiplicación de matrices (Fila de la primera * columna de la primera y se suma los valores) IMPORTANTE: Numero de filas de la primera = numero de columnas de la segunda
	- Rango: Número máximo de filas o de columnas independientes que tiene (haces el determinante y si sale 0 no puede ser ese rango y se prueba con uno menos)
			1 1 2
			1 1 3            El determinante es 0, y como el det de 1 2 es distinto de 0 su rango es2
			1 1 4                                                                           1 3
	- El rango se usa para indicar cuántas dimensiones son necesarias para describir completamente el objeto
	- Determinante: Valor escalar de la matriz, indica si la matriz tiene inversa o no
	- Inversa: Matriz que al multiplicarse por la matriz normal genera una matriz identidad 
	 A * A^-1 = Identidad, la inversa se utiliza para el cálculo de colisiones

- 1.1.5 Transformaciones afines y homogéneas:
	- Se usan para calcular las traslaciones, rotaciones y escalados de los objetos tridimensionales
	- Una transformación afín es una combinación de transformaciones lineales junto con la traslación, estas conservan las líneas rectas y la proporción, pero no tiene porque conservar ángulos o longitudes:
				x' = Ax + b    siendo A una mat 3x3 y b un vector de traslacion R3
	 - La traslación no es una traslocación afín
- 1.1.6 Coordenadas homogéneas:
	- Se usa cuando no se puede expresar algo con una matriz 3x3 y por ende se aumenta el tamaño del mismo (como en las traslaciones :))
	- La representación de un punto en este formato es (x,y,z,1) en lugar de (x,y,z)
	- Su matriz es: T = ( A b ) donde A es una matriz 3x3 y b es el vector de traslación 
					( 0  1 )
	- Este truco permite realizar cálculos contiguos de traslaciones
