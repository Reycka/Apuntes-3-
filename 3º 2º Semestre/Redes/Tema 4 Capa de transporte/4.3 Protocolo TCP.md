- Características:
	- Fases de transmisión:
		- 1. Establece conexión
		- 2. Transferencia de datos
		- 3. Cierre de conexión
	- Mecanismos de control de errores:
		- Numeración de segmentos
			- Cada segmento lleva un número de secuencia
			- Indica la posición que tiene el primer byte de datos del segmento del mensaje
		- Confirmaciones del receptor
			- Cuando le llega un segmento el receptor confirma la llegada
		- Retransmisión de segmentos:
			- Si no llega lo reenvía
	- ![[Pasted image 20260409093457.png]]
	- Flags:
		- URG: Puerto Urgente
		- ACK: Número de confirmación válido
		- PSH: Si los datos se deben pasar inmediatamente 
		- RST: Abortar conexión
		- SYN: Para establecer conexión
		- FIN: Finalizar conexión
