- Permite salir de mi red local a otra red o a internet, además permite saber si la máquina destino es de mi red local, básicamente encontrar el camino
- Métricas depende de la filosofía usan una u otra:
	- Encontrar el menor número de saltos
	- La menos distancia geográfica
	- Menor retardo promedio
- Tipos
	- Estático:
		- Las decisiones de encaminamiento consideran la topología de la red
		- Las tablas de encaminamiento se hacen a mano
		- NO se adaptan a los cambios en la red
	- Dinámico:
		- Las tablas se construyen solas mediante el intercambio de información
		- Se adpatan automáticamente a los cambios de red
		- Tecnicas:
			- RIP --> vector distancias
			- OSPF --> estado de los enlaces
			- BGP --> vector de rutas
- Cuando un router (encaminador) recibe un paquete lo retransmite por el enlace adecuado para alcanzar el destino

		- **A comunica a B:**
			- 1º A busca y encuentra a B en su tabla de encaminamiento

				- Mac destino B IP destino B Mac Origen A Ip origen A
			- El tipo del mensaje de A es ARP request (pregunta)
			- El tipo de mensaje de B es ARP reply
			- En cuanto a IP ping de A a B
			
		- **A comunica a C:**
			- 1º A busca y encuentra a C en su tabla de encaminamiento
			- 2º Al encontrarla comprueba si está en su red
			- 3º Como no lo está cambia la Mac destino a la del encaminador correspondiente
			- 4º El formato de cabecera IP del mensaje en sus 2 partes sería asi:
				- De A - Router2:
					- Mac destino R2 IP destino C Mac origen A IP origen A
				- De Router2 - C:
					- Mac destino C IP destino C Mac origen R2 IP origen A
			- El tipo del mensaje de A es ARP request (pregunta)
			- El tipo de mensaje de B es ARP reply 
			- En cuanto IP los pings al router de A y C y del router a ambos
			
		- **A comunica a D:**
			- 1º A busca a D en su tabla de encaminamiento

					- Mac destino R1 IP destino D Mac origen A IP origen A
				- Router1 - D 
					- Mac destino D IP destino D Mac origne R1 IP origen A
			- El tipo del mensaje de A es ARP request (pregunta)
			- El tipo de mensaje de B es ARP reply
