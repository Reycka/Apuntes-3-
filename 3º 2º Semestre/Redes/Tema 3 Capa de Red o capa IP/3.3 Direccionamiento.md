- Estructura y notación:
	- Las IPS constan de 4 bytes (32 bits)
	- Se utiliza la notación punto a punto (10.24.2.0)
	- ![[Pasted image 20260219091346.png]]
	- Tipos:
		- Uncast: La dirección identifica un único host
		- Broadcast: La dirección identifica todos los hosts posibles 
		- Multicast: La dirección representa un grupo de hosts
- Administración y registro de direcciones:
	- a
	- El nombre tiene un número reservado de bits al principio (dependiendo de la clase el número cambia) 
	- ![[Pasted image 20260219091236.png]]
	- ¿Qué pasa si tenemos una empresa que quiera un número X de hosts pero que no se adapte a ninguna de las clases?
		- Propuesta: ![[Pasted image 20260219092359.png]]
		- Solución: DIRECCIONAMIENTO SIN CLASES
			- El host y la red nos están sujetas a un número específico de bytes
			- ![[Pasted image 20260219092613.png]]
			
	- Direcciones especiales:
		- Son direcciones reservadas para uso privado
		- No se pueden conectar a internet pero:
			- Se pueden asignar a redes aisladas de internet
			- Se pueden asignar a redes a través de un router (ES EL ROUTER QUIEN ACCEDE A INTERNET NO NOSOTROS)
		- Rangos de direcciones privadas:
			- 10.0.0.0 - 10.255.255.255 --> 1 red privada de clase A
			- 172.16.0.0 - 172.31.255.255 --> 16 redes privadas de clase E
			- 182.168.0.0 - 192.168.255.255 --> 256 redes privadas de clase C
		- Loopback
			- Direcciones que referencian al propio dispositivo
			- No se envían por la red
			- Su formato es 127.0.0.1
		- Direcciones reservadas:
			- TODO 1 en el host: reservada para el broadCasat
				- ![[Pasted image 20260219093829.png]]
			- TODO 0 en el host: reservada para el nombre de la red
				- ![[Pasted image 20260219093919.png]]
				
	- Máscara de red:
		- La máscara de red sirve para indicar cuantos bits de la dirección IP identifican a la red y cuantos identifican al host dentro de la red, se usa porque el formato usado es AND de forma que SIEMPRE y digo SIEMPRE me da la red en concreto (pq el 1 de la red no me cambia valores) y me elimina el puesto dejándome solo el nombre de la red (pq el 0 del host se carga todos los valores pq los elimina)
		- FORMATO:
			- 0 bits del host
			- 1 bits de la red
		- ![[Pasted image 20260219094304.png]]
		- Alternativa: NOTACIÓN CIDR 221.98.22.2/24
		- El valor /24 indica la longitud de la parte de red (nº de unos de la máscara)
		- Cada interfaz tiene su IP y su máscara, ya que si no jodienda tocha no se puede trabajar
