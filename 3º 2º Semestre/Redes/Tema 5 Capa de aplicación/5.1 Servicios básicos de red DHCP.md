- Características:
	- Configura automáticamente los parámetros de la red
		- Dirección IP, máscara, router predeterminado, servidor DNS
	- Clientes DHCP:
		- No tienen una configuración de red fija
		- Al arrancar el sistema se busca un servidor DHCP que le proporcione la información de red necesaria
	- Servidor DHCP:
		- Proporciona los parámetros de configuración de la red a los clientes que lo solicitan
		- Es utilizado en redes locales para asignar IP dinámicas y otros parámetros de red a los dispositivios conecctados
		- Acceso telefónico o ADSL a través de ISP
	- Usan UDP NO SE USA TCP
- Funcionaminento: (El mítico DORA)
	- 1º Discovery: El cliente envía broadcast DHCPDISCOVER para buscar los servidores DHCP activos
	- 2º Offer: El servidor responde con un DHCPOFFER con una oferta de parámetros de configuración conforme a la situación del cliente
	- 3º Request: El cliente pregunta por la oferta en cuestión, luego el servidor puede decidir si aceptarlo o rechazarlo (2 posibles mensajes)
	- 4º Acknowledge (confirmacion): El servidor envía un mensaje de confirmación
	- 5º El clente al marcharse manda un DHCPRELEASE
