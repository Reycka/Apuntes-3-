- Definiciones:
	- Ciberseguridad: Protección al sistema informático, tratando de evitar el número de ataques o comportamientos dañinos.
		- Para ello se intenta prevenir el ataque con varias medidas **preventivas**, en caso de que ataquen y consigan entrar al sistema tenemos que tener un sistema de **detección** que comprueben si hay algo fuera de lo normal. Si pese a todo no se ha podido evitar el ataque y ha habido daños, se debe poder **reaccionar** para arreglar cuanto antes mejor e intentar revertir ese daño
	- Seguridad como gestión: No sabes como ni cuando te van a atacar ni los riesgos que eso conlleva. "*Haz que el adversario trabaje más que tú*". Si quiere entrar que se lo curre al menos
		- Preguntas clave: ¿A quién? ¿Qué? ¿De quién? ¿Cómo? y ¿Cuándo?, "*Si no es Simple no es seguro*"
			- ¿QUÉ?: Nuestros datos (confidencialidad), las máquinas (disponibilidad), el Software (disponibilidad, integridad) y las redes (disponibilidad)
			- ¿A QUIÉN? Nuestros clientes, datos dinero estado psicológico... y nuestro negocio, derechos de autor, código fuente y cumplimiento legal
			
- Pilares de la ciberseguridad:
	- CONFIDENCIALIDAD
	- INTEGRIDAD --> Que el contenido sea real, no sea una modificación del contenido
	- AUTENTICIDAD --> Garantizar que quien dice ser es en realidad
	- DISPONIBILIDAD
	
- Las redes cibercriminales funcionan como una empresa, departamentos muy organizados y son personas con gente especializada en cada una de las partes del hackeo, buscan datos que puedan vender por mucho dinero 
- El ransomware es un troyano o virus (depende de como entra) que cifra toda la información del sistema y solo con la clave que tiene el hacker se puede desencriptar el sistema (es como un secuestro que se pide un rescate, pero ahora es un sistema)
- Malware:
	- Programa informático que se activa sin conocimiento ni autorización del propietario infectado con el objetivo de realizar varias funciones perjudiciales
	
		- Virus: Programa malicioso que busca replicarse y pasarse de un dispositivo a otro, con el objetivo de dañar el sistema, robar información etc.
		- Gusanos: Virus que puede replicarse entre ordenadores, a menudo causa errores en la red por consumo de ancho de banda
		- Bombas lógicas: Código oculto que no se ejecuta hasta X momento
		- Troyanos: Programa que se oculta dentro de otro con un código que ejerce cosas no deseadas
		- Puerta trasera: Partes del código que permitne entrar a alguien sin credenciales ni autentificación
		- Exploits: medio de infección, ejecutan el malware NO SON EL MALWARE
		- Rootkits: Oculta el virus para que este no sea encontrado
