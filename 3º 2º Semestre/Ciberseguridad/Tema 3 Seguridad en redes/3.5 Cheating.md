- Definiciones:
	- Cheating: Un jugador no sigue las reglas que se establecen en el juego para ganar:
		- Tipos:
			- Complots (Hacer team)
			- Compartir cuentas
			- Secuestro de cuentas
			- Uso de bots
			- Manipulación HW
			- Editores de memoria
			- Vídeos falsos 
	- Exploiting: Explotar fallos de un videojuego que sus desarrolladores no han detectado
		- Cheat codes (código de testing que no se eliminan en la versión final)
		- Duplicación
		- Geometría (Acceso a áreas imposibles )
		- Twinking / Smurfing (Equipo muy bueno con muy poco nivel)
		- Desconexión (Manipular la conexión para conseguir ventaja) entra dentro de ataques DoS (Denegación de servicios)
		- Bugs de velocidad de movimiento
- Clasificación por capas:
	- Sin manipular el SW cliente
	- Manipular el SW cliente
	- Manipular SW/HW local externno al SW cliente
	- Manipular las comunicaciones con el servidor / con otros clientes
	- Manipular el servidor
	- Mundo real
- Que el servidor reciba la información pero al cliente no le de tiempo a cambiarla y justo el tramposo se sale o lo que sea entonces se queda con las cosas de antes o la info de antes pero el servidor manda a los demás usuarios la info real (útil en intercambio y duplicación de objetos)

- Para detectar BOTS se pueden aplicar medidas tales como:
	- Captha automático
	- Buscar colaboración con los jugadores del juego
- Detección de complots
	- Métodos estadísticos que detectan que el comportamiento de varios jugadores siempre benefician a los mismos 
- DoS
	- La auto-desconexión para encontrar algún beneficio
	- Denegación de servicios al servidor por inundación (muchos pings)
		- Un firewall con un minimo de pings que pueda recibir por IP
		- Si es hacia el cliente no tengas en el cliente las IPS de los demás jugadores, no es información mínima necesaria
	- Tirar del cable (detectar cuando la desconexión resulta ventajosa) confidencialidad
